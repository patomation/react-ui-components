## Collapse
A component that can collapse and hide content

### Props:

#### className
Exposes ability to set a custom class name
 - type: string

#### children
 - type: union
   - type: array of node
   - type: node

#### style
 - type: object

#### collapse
 - type: bool

#### onComplete
 - type: func

#### speed
 - type: number
 - default: 400
