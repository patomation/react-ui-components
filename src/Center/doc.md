## Center
Aligns any content to the center both vertically and horizontally

### Props:

#### className
Exposes ability to set a custom class name
 - type: string

#### children
 - type: union
   - type: array of node
   - type: node

#### background
 - type: string

#### color
 - type: string

#### style
 - type: object

#### width
 - type: string

#### maxWidth
 - type: string

#### disabled
 - type: bool
 - default: false
