## Clickable
Wraps anything and makes it clickable

### Props:

#### className
Exposes ability to set a custom class name
 - type: string

#### children
 - type: union
   - type: array of node
   - type: node

#### onClick
 - type: func

#### background
 - type: string

#### color
 - type: string

#### style
 - type: object

#### disabled
 - type: bool
