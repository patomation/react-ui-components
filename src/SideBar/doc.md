## SideBar
a layout that makes a side menu and a content area

### Props:

#### className
Exposes ability to set a custom class name
 - type: string

#### children
 - type: union
   - type: array of node
   - type: node

#### menu
 - type: union
   - type: array of node
   - type: node

#### style
 - type: object

#### menuStyle
 - type: object

#### contentStyle
 - type: object

#### width
 - type: string
 - default: '200px'

#### right
 - type: bool
